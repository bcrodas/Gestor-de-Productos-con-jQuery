Creación de una mini-aplicación web utilizando jQuery y Bootstrap 5, funcionando completamente en entorno local.
El propósito de esta actividad es aplicar y reforzar los conocimientos sobre:

Estructuras de control
Estructuras iterativas
Arrays y objetos
Manejo de eventos con jQuery
Manipulación del DOM
Componentes básicos de Bootstrap 5

La aplicación debe permitir agregar productos, listarlos, filtrarlos, ordenarlos y mostrar estadísticas, utilizando jQuery como herramienta principal de programación.
